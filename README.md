# IFT3225-tp1
Repo de code pour le TP1 du cours IFT3225 technologies de l'internet.
### Sujet
Application web utilisant la stack LAMP. L'application permet l'ajout d'exercices pour la creation personnaliser d'entrainement.

Inspirer de l'application mobile [Hevy](https://www.hevyapp.com/)

L'utilisateur peut ajouter de nouveaux exercices au systeme. 
L'utilisateur peut modifier et supprimer les exercices qu'il a ajouter a l'application.
